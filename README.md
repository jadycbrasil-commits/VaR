# Value at Risk (VaR)

Projeto desenvolvido para demonstrar conhecimentos em:

- SQL Avançado
- Window Functions
- Estatística Financeira
- Gestão de Risco de Mercado
- Python para análise quantitativa

Métodos implementados:

- VaR Histórico
- VaR Paramétrico
- Cálculo de Retornos
- Visualização de Distribuição de Risco
